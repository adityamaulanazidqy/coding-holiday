# Coding Holiday

Project ini merupakan kumpulan latihan koding yang dibuat oleh kakak-kakak asisten laboratorium. Daripada waktu liburan terbuang cuma-cuma atau gabut, lebih baik digunakan untuk mengasah logika dan kemampuan pemrograman melalui berbagai tantangan koding setiap minggunya.

## Daftar Isi

1. Minggu 01: Sistem Absensi (Struktur dasar dan I/O)
2. Minggu 02: Input Data Mahasiswa (Manipulasi Pointer dan Struct)
3. Minggu 03: Latihan Struct dan Pointer (Studi kasus lanjutan)

## Cara Menjalankan

Setiap folder memiliki Makefile masing-masing. Untuk menjalankan program, masuk ke folder yang diinginkan lalu jalankan perintah:

```bash
make
make run
```

Untuk membersihkan file eksekusi:

```bash
make clean
```
